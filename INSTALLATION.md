# Installation

## With requirements.txt

```
git clone https://github.com/Avaiga/demo-stock-visualization.git
cd demo-stock-visualization
cd src
pip install -r requirements.txt
```

## With Pipfile

Please install `Pipenv`:
```
pip install pipenv
git clone https://github.com/Avaiga/demo-stock-visualization.git
cd demo-stock-visualization
pipenv install
```
